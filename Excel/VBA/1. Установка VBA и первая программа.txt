Для того, чтобы перейти в VBA нужно включить опцию разработчика.
Для этого - мы переходим в Файл -> Параметры -> Настроить ленту -> Ставим галочку напротив Разработчик

Далее у нас появляется вкладка разработчик, мы в неё переходим и нажимаем на Visual Basic, мы попадаем в окно VBA.

Далее настроим VBA.

Перейдём в Tools -> Options и смотрим настройски с рисунка 1.
Жмём Ок.

Далее создадим нашу первую программу. Для этого запустим нужно создать модуль. Insert -> Module.
И напишем программу Hello World!

Вставляем данный код

Sub testSub()
    MsgBox "Hello World!"
End Sub

Запускаем с помощью кнопки Run на панели интрументов. Он может попросить выбрать какую из функций запускаем.
Выбираем нашу созданную то есть helloSub
Готово.
