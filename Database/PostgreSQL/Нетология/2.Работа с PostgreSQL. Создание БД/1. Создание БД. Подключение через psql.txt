Для создание базы данных - в терминале воспользуемся следующей командой

createdb -U postgres test

где:
createdb - выполняемая команда
-U postgres - указание того что мы подключаемся от имени пользователя postgres
test - название БД

И теперь подключимся к бд, которой хотим управлять
с помощью команды:

psql -U postgres -d test

psql - выполняемая команда
-U postgres - от чего имени мы хотим подключится
-d test - какой БД мы хотим управлять

После входа в режим управления БД 
Мы можем использовать символы \?
Для получения полного списка команд

Для того чтоб удалить БД нужно воспользоваться командой

dropdb -U postgres test

Синтаксис аналогичен двум прошлым командам
