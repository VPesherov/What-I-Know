План занятия:
1. Концепт ORM
2. Абстракция ORM

ORM(объектно-реляционное отображение) - это дополнительный способ взаимодействия с БД из
кода, который работает с таблицами и запросами к БД как с классами, объектами и методами
в ООП.

Для Python - одна из самых популяных ORM - это SQLAlchemy
Для установки используем команду
pip install sqlalchemy

Итоги:
ORM - это дополнительный способ взаимодействия с БД из кода, который работает с таблицами и 
запросами к БД как с классами, объектами и методами в ООП

ORM можно использовать и для существующих БД, и при создании новой БД.
