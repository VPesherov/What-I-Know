Применения issues можно найти на Рисунке 1.
Для того чтоб добавить issues в удалённом репозитории на github переходим в раздел
issues

Далее нажимаем на кнопку 
New issue

Далее заполняем название задачи и подробное описание то что нужно сделать или с чем вы столкнулись
,например, описать баг или что хотите добавить.
Перейдя в окно
Preview

Увидим предпросмотр того как будет выглядить наш issue.
Также через символ 
@

Можно в описании issue отметить других пользователей.
Справа есть настройки.
В поле
Assigness

Можно назначить исполнителя этой задачи.
В label

Можно поставить тег проблемы с которой вы столкнулись.
Нажимаем 
Submit new issue
