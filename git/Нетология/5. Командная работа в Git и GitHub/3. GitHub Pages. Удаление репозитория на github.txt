GitHub pages - это удобный бесплатный хостинг для статических файлов, созданных на базе HTML, css, JS и Markdown.
Также его удобно использовать для создания портфолио.

Для публикации в нашем репозитории на github - нужно перейти во вкладку 
settings

В левой колонке - найдите пункт 
pages

Далее в выпадающем списке source выбираем
Deploy from a branch

Далее под заголовком Branch выбираем ветку 
main

Также можно выбрать папку, которую будем демонстрировать и после того - как всё выберем можно нажать кнопку
save

Далее при обновлении страницы - появится кнопка
Visit site

Также можно выбрать тему для сайта нажав на кнопку
Choose theme

Чтоб удалить репозиторий с github - нужно перейти снова в 
settings

Затем пролистать в самый низ и нажать кнопку
delete repository
