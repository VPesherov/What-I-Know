Что делать если накосячили с коммитами в удалённом репозитории?
Разберёмся.
В этом поможет команда 
git revert

Узнать подробнее о сути работе git revert и его плюсах можно узнать на Рисунке 1.
Чтоб начать удалять коммит, для начала нужно выбрать коммит и скопировать его id
Введём команду
git log --oneline

и копируем последовательность слева нужного коммита, это будет что-то типо
83e3073

После выполнения 
git revert 83e3073

Нужно будет название для коммита отмены.
