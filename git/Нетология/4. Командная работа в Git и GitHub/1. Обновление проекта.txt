Обновление проекта нужно, чтоб иметь актуальную версию. Так как другое разработчик мог уже внести какие-то изменения.
Чтоб получить изменения другого разработчика нужно выполнить команду
git pull

Затем система git может открыть окно текстового редактора, чтоб вы назавали commit, которые зафиксирует то что вы 
сделали pull веток.
Можно просто выйти из текстового редактора и тогда commit останется таким же.
