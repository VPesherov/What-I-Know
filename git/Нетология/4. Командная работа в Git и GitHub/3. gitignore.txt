Базовая информация о gitignore показана на Рисунке 2.
Допустим у нас есть папка, которую мы не хотим добавлять в удалённый репозиторий.
Решим эту проблему с помощью gitignore
Создаём файл с название 
.gitignore

Чтоб игнорировать целую папку, пишем в этот файл
Название_папка/

Теперь если мы введём команду
git status

Мы не увидем папку, которую добавили в файл gitignore
