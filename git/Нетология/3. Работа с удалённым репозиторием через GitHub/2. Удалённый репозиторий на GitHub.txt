Для создания собственного репозитория на GitHub нужно нажать на 
+ в правом верхнем углу и выбрать
New Repository

Проверяем, чтоб было выставлено
Public

Остальное добавлять пока не надо
Далее переключаемся в режим ssh и копируем ссылку

И теперь попробуем связать нашу локальную папку test с удалённым репозиторием test
Чтоб убедиться что у папки нет связи с удалёнными репозиториями нужно ввести команду
git remote -v

Если консоль ничего не выдала, то папка не связана
Чтоб связать выполним похожую команду
git remote add origin (вставляем ссылку, которую скопировали до этого)

Теперь если введём git remote -v
Получим что-то типо

origin  git@github.com:VPesherov/test.git (fetch)
origin  git@github.com:VPesherov/test.git (push)

Теперь попробуем отправить что-то из нашей папки в удалённый репозиторий на github
Для этого нужна команда
git push -u origin main

Если что-то пошло не по плану, значит нужно попробовать пересоздать ssh ключ
Если успешно выведется что-то типо этого
Branch 'main' set up to track remote branch 'main' from 'origin'.

Теперь обновим страницу нашего репозитория на github и увидем наши файлы.
