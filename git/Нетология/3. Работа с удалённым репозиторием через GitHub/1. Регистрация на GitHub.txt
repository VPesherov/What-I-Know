Перед регистрацией, для начала нужно уметь различать Git и GitHub.
Разницу можно увидеть на Рисунок 1.

Мы выбираем GitHub - так как он является самым популярным.
Для регистрации - заходим на сайт github.com и жмём кнопку sign up(регистрация)

-Вводим email
-И пароль
-Придумываем оригинальное имя пользователя
-Далее спросят о том, хотим ли мы получать рассылку на почту? жмём y или n
-Далее проходим проверку на робота бип боп
-Создаём аккаунт
-Вводим код с почты
-Далее отвечаем на вопросы (можно пропустить)
-Профиль создан!

Далее нужно создать SSH-ключи, он помогает создать такой способ авторизации, который поможет синхронизировать ваш компьютер
и профиль в сети.
Зачем он нужен можно посмотреть на Рисунок 2.

Чтоб создать ssh-ключ нужно выполнить следующие команды, пишем:
ssh-keygen

Далее можно просто несколько раз нажать enter
Далее ключ сгенирируется, если всё успешно - в тексте можно будет найти строку
You public key has been saved in

В конце данной строчки будет путь, нам нужно скопировать часть этого пути начиная с
/.ssh/id_rsa.pub

Дальше в терминале вводим команду
cat ~и ту фразу которую скопировали

Терминла ответит большим набором цифр и символов. Нужно его скопировать.
Далее возвращаемся на github и октрываем наш профиль -> settings -> SSH and GPG keys
Нас интересует пункт 
SSH keys
Нажимаем New SSH key

Вводим в поле 
title

Любое название для ssh ключа 
Далее в большое поле вставляем ssh ключ, который скопировали из терминала, затем жмём 
Add SHA key

!!! Если мы переходим на другой компьютер, то ключ нужно генерировать и прикреплять заново.
