Тут мы решим проблему, как склонировать проект на свой пк?
Это называют Fork, что это такое конкретно можно посмотреть на Рисунке 3.

Чтоб создать Fork - нужно на GitHub в правом верхнем углу - нажать 
Fork 

и далее нажать
Create fork

Теперь можно увидеть, что у вас есть личная копия данного проекта, в левом углу под названием репозитория будет вставлено ваше имя пользователя.
Под этой строкой будет подпись 
forked from откуда-то

Теперь нам нужно склонировать репозиторий для этого жмём кнопку 
Code

Далее переходим на вкладку 
SSH

И копируем ссылку.
Открываем терминал в нужном месте и пишем команду
git clone ссылка

Произойдёт клонирование проекта на компьютер и будет создана новая папка в которую нужно не забыть перейти.
