После внесения изменения в файл и вызова комнады git status
мы увидем статус у файла modified

Чтоб увидеть краткую историю изменений напишем команду 
git log --oneline

Если хотим увидеть полную историю проекта то нужно просто ввести
git log

Если допустим мы допустили ошибку в описание коммита то мы можем воспользоваться следующей командой
git commit --amend -m "Правильное название"

Если мы вдруг забыли добавить что-то в предыдущий коммит, а новый создавать не хочется то можно тоже воспользоваться командой
git commit --amend -m "Название"

Но перед этим не забудьте добавить файлы в отслеживаемые
В итоге вместо двух коммитов у нас будет один
