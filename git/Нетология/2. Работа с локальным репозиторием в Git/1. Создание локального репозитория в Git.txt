Что такое репозиторий?
Репозиторий - это папка за которой следит git.

Есть два вида репозиториев - локальный, который находится у вас на компьютере и удалённый репозиторий, который находится
где-то в облаке.

Чтоб создать репозиторий, нужно перейти в эту папку в терминале и ввести команду 
git init
, если всё успешно то в этой папке можно будет увидеть папку .git

Принцип работы git - можно увидеть на рисунке 1.

Файлы могут находится в двух состояниях в репозиториях. Отслеживаемые(с котнролем со стороны git) и неотслеживаемые(без контроля
со стороны git).

Чтоб проверить отслеживаемые и неотслеживаемые файлы, нужно ввести команду
git status

Чтобы начать отслеживать какой-либо файл нужно ввести команду 
git add Название_файла

Теперь зафиксируем изменения в данном файле с помощью команды 
git commit -m "Название изменений" 
*чтоб сделать две команды одновременно git add + git commit можно воспользоваться следующей командой
git commit -a -m "Название изменений"

Принято называть первый коммит - Initial commit

Краткий план по созданию репозитория будет на рисунке 2.
По фиксированию изменений на рисунке 3.
Чтоб посмотреть что git видеть проверяем командой git status
