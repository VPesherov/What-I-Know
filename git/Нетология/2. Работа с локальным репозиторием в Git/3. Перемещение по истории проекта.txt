Чтоб перейти к нужному коммиту в начале выведем все коммиты в коротком виде
git log --oneline

И скопируем код стоящий перед названием коммита
И попробуем перейти на другой commit
Для этого воспользуемся коммандой
git checkout Код

Теперь если ввести
git log --oneline

То мы увидем, что изменения которые мы делали после не отображаются
Чтоб всё же увидеть коммиты которые были до этого нужно ввести команду
git log --oneline --all

Указатель HEAD будет указывать, где мы сейчас находимся
Чтоб вернуться обратно нужно ввести либо
git checkout main 
или
git checkout master

В итоге мы переместимся на последние изменения
