Допустим мы хотим сравнить два объекта по каким-то критериям
Проверить помогут специальные магические методы
Справа операции за которые они отвечают

__eq__ # ==
__ne__ # !=
__lt__ # <
__le__ # <=
__gt__ # >
__ge__ # >=

Создадим класс прямоугольник

class Rectangle:

    def __init__(self, a, b):
        self.a = a
        self.b = b

    def __eq__(self, other): # метод __eq__ проверяем равенство двух экземпляров
        if isinstance(other, Rectangle):
            return self.a == other.a and self.b == other.b

a = Rectangle(100, 200)
b = Rectangle(100, 200)
c = Rectangle(200, 300)
a == b # True
b == c # False

Вот так мы легко проверили равенство двух прямоугольников другие функции реализуются аналогично
