Создадим класс Car с атрибутами model и engine

class Car:
    model = "BMW"
    engine = 1.6

и 2 экземпляра класса

a1 = Car()
a2 = Car()

и у каждого экземпляра класса есть его атрибуты

a1.model # BMW
a2.engine # 1.6

Присвоим одному из экземпляров класса новый атрибут
a1.seat = 4

в итоге он создасться только в данном экземпляре в a2 его не будет
a2.seat # будет ошибка

и этот атрибут не будет и в основном классе

Car.seat # будет ошшибка
только a1.seat # 4
