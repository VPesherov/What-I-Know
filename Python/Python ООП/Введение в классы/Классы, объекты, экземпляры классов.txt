Что такое объект в программировании?

Объект - контейнер, состоящий из
1)данных(атрибуты) и состояний
2)поведения(методы)

Пример:

Есть список a = [5, 3, 34] и данными(атрибутами) будут являться сами элементы списка
а поведением будут являться методом списка например a.append(1)

Каждый объект принадлежит определённому классу и это можно проверить с помощью type
type(a) # list

Создание простого класса

class Car:
	pass

теперь присвоим переменной класс Car 
a = Car()
и посмотрим его тип
type(a) # __main__.Car

Определив один раз класс - мы можем создавать множество объектов
b = Car()
Проверим
isinstance(b, Car) # True

Создадим класс чуть посложнее, а точнее сказать переопределим класс который у нас был

class Car:
    model = 'BMW'
    engine = 1.6

и создадим экземпляр класса c

c = Car()

Теперь в c есть информация о model и engine из класса Car
