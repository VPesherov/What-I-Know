Терминал - это программа, которая позволяет управлять компьютером посредством текстовых команд.

Зачем вообще нужен терминала если есть десктоп приложения?
1. Некоторые действия - можно сделать только из командной строки
2. Серверы, на которые размещают проекты - чаще всего далеко от вас - и подключаться к ним чаще всего можно будет - только
посредством терминала
3. Конвейер

Задача:
У нас есть меню в .txt - с повторяющимися строками

Рыба
Перловка
Гречка
Перловка
...
и тд

Необходимо выяснить - сколько каких блюд заказали и отсортировать по убыванию.
Решение - может состоять из одной команды в строке

cat menu.txt | sort | uniq -c |sort -r

Результат будет:
2 Перловка
1 Гречка
1 Перловка

На самом деле - это не одна команда - а это тот самый конвейер команд

bash - мы и будем изучать так как чаще всего в разработке используют именно его.

Команды и ключи

Команды - это подпрограммы, аналоги функций в python, бывают с параметрами, бывают без.
Например - подпрограмма вывода содержимого файла 

cat имя_файла

Сам python - это тоже программа, которую запускает интерпретатор

q - Выход из окна
Ctrl + c  - выйти из процесса
home или end - переместить курсор
