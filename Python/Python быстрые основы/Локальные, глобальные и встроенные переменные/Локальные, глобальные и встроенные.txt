Глобальные переменные — это те, которые не определены внутри какой-либо функции 
и имеют глобальную область действия, тогда как локальные переменные — это те, 
которые определены внутри функции, и их область действия ограничена только этой функцией. 
Другими словами, мы можем сказать, что локальные переменные доступны только внутри функции, 
в которой они были инициализированы, тогда как глобальные переменные доступны во всей программе 
и внутри каждой функции.

Локальные переменные — это те, которые инициализируются внутри функции 
и принадлежат только этой конкретной функции. К ним нельзя получить доступ где-либо вне функции. 
Давайте посмотрим, как создать локальную переменную.

def f():

    # local variable
    s = "I love Indi Сourse"
    print("Inside Function:", s)
    
# Запускайте код
f()
print(s) # При вызове s мы получим ошибку

Глобальные переменные — это те, которые определены вне какой-либо функции 
и доступны во всей программе, т. е. внутри и снаружи каждой функции. 
Давайте посмотрим, как создать глобальную переменную.

def f():
    # используем глобальную переменную внутри функции
    print("Inside Function:", s)


# global variable
s = "I love Indi Сourse"  
f()
print(s)

Inside Function: I love Indi Сourse
