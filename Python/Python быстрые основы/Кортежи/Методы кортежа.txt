Метод index() - показывает первый слева индекс элемента, если он присутствует в кортеже:

a = (1, 2, 3, 54, 8, 6, 2, 7)
print(a.index(54)) #3
print(a.index(3)) # 2
print(a.index(2)) # 1


Метод count() выполняет подсчёт количества вхождений данного элемента в кортеж:

a = (1, 2, 3, 54, 8, 6, 2, 7)
print(a.count(54)) # 1
print(a.count(99)) # 0
print(a.count(2)) # 2
