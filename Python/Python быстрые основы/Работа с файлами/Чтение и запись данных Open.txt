В файлах хранится основная информация, необходимая для работы. 
Поэтому необходимо научиться считывать данные с файлов и записывать их туда.
Для открытия файлов существует специальная функция open,

def open(file, mode='r', buffering=None, encoding=None, errors=None, newline=None, closefd=True)
Она может принимать несколько аргументов, но только один из них обязательный - file. 
При помощи параметра file указывается путь к файлу и его название.

Если ваш файл лежит в той же папке же папки, где запускается программа на python, то в параметр 
file достаточно передать только название файла.

К примеру создайте текстовый документ с названием my_text.txt. 
Можете скопировать туда следующее содержимое:

Я очень хочу, чтобы людей,
умеющих программировать на Python
было больше, поэтому мне
необходима ваша поддержка!
Подписывайтесь мой на канал!
https://www.youtube.com/channel/UCMcC_43zGHttf9bY-xJOTwA

В этой же папке создайте питоновский модуль и воспользуйтесь функцией open

my_file = open('my_text.txt')
После этого в переменной my_file будет хранится объект TextIOWrapper, который может выполнять 
различные действия над файлами. 
Например, через метод .read можно прочитать все содержимое файла.

my_file = open('my_text.txt')
print(my_file.read())

# Вывод
Я очень хочу, чтобы людей,
умеющих программировать на Python
было больше, поэтому мне
необходима ваша поддержка!
Подписывайтесь мой на канал!

Если вы видите в выводе непонятные символы, это значит функция open не смогла правильно 
раскодировать кириллицу. Чтобы помочь ей правильно раскодировать правильно русские буквы, 
нужно передать название кодировки utf-8 в аргумент encoding

my_file = open('my_text.txt', encoding='utf-8')
print(my_file.read()) # теперь вывод будет на русском

Если файл находится не в этой же директории что и питоновский модуль, то нужно указать абсолютный путь
и лучше этот делать через параметр r

file = open(r'C:\Users\admin\PycharmProjects\pythonProject3\my_text.txt', encoding = 'utf-8')
print(file.read())
