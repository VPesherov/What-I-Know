два множества называются равными или эквивалентными, если они состоят из одних 
и тех же элементов и имеют одинаковое количество элементов. 
Как следует из определения множества, порядок этих элементов не важен. 
Для проверки множеств на равенство используется уже знакомый вам оператор ==

a = {1, 2, 3}
b = {3, 1, 2}
print(a == b)
print(a != b)

print(a == {1, 2})

# Вывод
True
False
False
#
