Теперь рассмотрим следующий пример:

def f(a, b, c):
    print(a, b, c)
Здесь мы определи функцию f  с тремя параметрами. 
Существует несколько способов вызвать эту функцию и передать в нее аргументы. 
Давайте с ними познакомимся:

1) Позиционный. Указать название функции и передать напрямую три значения. 
В таком случае они будут присваиваться в том же порядке, что и стоят. 
Пример:

def f(a, b, c):
    print(a, b, c)
    
f(1, 5, 7) # 1 5 7


2) По имени. При вызове функции указывается имя переменной и необходимое ей значение.

def f(a, b, c):
    print(a, b, c)
    
f(b=20, c=15, a=5) # 5 20 15

3) Комбинированный вариант совмещает в себе передачу по позиции и по имени

def f(a, b, c):
    print(a, b, c)
    
f(2, c=10, b=20) # 2 20 10

Важно учитывать то, что сначала указываются позиционные аргументы, 
а потом именованные. Подобная запись привела бы к ошибке:

def f(a, b, c):
    print(a, b, c)
    
f(c=10, b=20, 2) # Ошибка
