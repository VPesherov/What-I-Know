Синтаксис функции

def имя_функции(параметры):
   <тело функции>

Здесь стоит выделить несколько особенность этого шаблона:

1) Объявление функции всегда начинается с ключевого слова def

2) За ним следует имя вашей функции, его вы придумываете сами. 
Название должно быть уникальным, иначе будет конфликт имен

3) После названия функции обязательно открываются круглые скобки, 
где перечисляются имена входных параметров вашей функции. 
Параметров может быть сколько угодно и может их не быть вовсе

4) После закрытия скобок, где перечисляются параметры, обязательно ставиться двоеточие

5) После двоеточия на новом уровне отступов пишется тело функции - это тот самый набор инструкций, 
объединенных для общей задачи. Туда и надо писать код, отвечающий за всю логику работы функции.
