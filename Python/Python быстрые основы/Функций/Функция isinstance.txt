функция помогает определить тип переменной например

синтаксис isinstance(проверяемая_переменная, тип)

Пример:

a = [5, 3, 'hello', [3, 4], 'world', [5], 10.5]

for i in a:
    if isinstance(i, str):
        print(i)

# Вывод
hello
world
#

то есть если к нам попала строка, то мы её выводим
это удобно если к нам поступает большое количество данных и их все нужно проверить

а если мы одновременно хотим проверить данные на несколько типов сразу то тогда
можно передать кортеж из типов например мы хотим считывать числа и типа int и типа float

total_sum = 0
for i in a:
	if isinstance(i, (float, int)):
		total_sum += i

print(total_sum) # 18.5
