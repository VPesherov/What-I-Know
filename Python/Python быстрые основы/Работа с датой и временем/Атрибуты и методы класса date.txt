Атрибут max

При помощи атрибута max вы можете получить значение максимальной даты. 
Обращаться к этому атрибуту можно как через сам класс date, так и через переменные, 
в которых хранятся даты

from datetime import date
print(date.max)

a = date(2015, 8, 21)
print(a.max)

# Вывод
9999-12-31
9999-12-31
#

Метод today

При помощи него можно получить текущую дату. Не забудьте его вызвать (поставить круглые скобки)

from datetime import date

now = date.today()
print('Сегодня:', now) # Сегодня: 2022-11-16

Метод weekday

Метод weekday дает порядковый номер дня недели. При этом данный метод считает следующим образом:

понедельник = 0
вторник = 1
среда = 2
четверг = 3
пятница = 4
суббота = 5
воскресенье = 6

Метод isoweekday также возвращает порядковый номер дня недели, только в нем отсчет 
идет с единицы: понедельник - 1, вторник - 2 ... воскресенье - 7

from datetime import date

now = date.today()
print('Сегодня:', now)
print('День недели:', now.weekday())
print('День недели:', now.isoweekday())

# Вывод
Сегодня: 2022-11-16
День недели: 2
День недели: 3
#
