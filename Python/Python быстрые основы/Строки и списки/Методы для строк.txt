все буквы строки вверх .upper()
Пример
'hello'.upper()
# 'HELLO'

все буквы строки вниз .lower()
Пример
'HEllO'.lower()
# 'hello'

Посчитать количество вхождения какой-либо буквы в строку
Пример
s = 'hello world'
print(s.count('l'))
#Вывод 3
Также для метода count есть свой срез
s = 'hello world'
print(s.count('l', 2, 7))
# сколько раз встретися буква 'l' в строке со 2-го по 6й символ включительно

Найти индекс нужного элемента .find()
s = 'hello world'

print(s.find('o'))
# Вывод 4
print(s.find('x'))
# Вывод -1 так как x нет в слове

Метод для перезаписи какой-либо буквы replace()
s = 'hello world'
print(s.replace('e', 'a'))
Тут есть необязательный третий параметр, он указывает на максимальное количество замен которые
надо сделать в тексте
print(s.replace('l','?', 2))
# Вывод he??o world # последняя l не меняется

Метод который проверяет состоит ли строка только из букв isalpha()
s = 'hello world'
print(s.isalpha())
# Вывод False потому что в строке присутствует пробел

Проверка являвется ли строка числом isdigit()
s = '123'
print(s.isdigit()) 
# Вывод True

Дополнение строки если она меньше какого-то значения ljust() и rjust()
d = '111'
print(d.rjust(5)) # Вывод __111 , так как строка d меньше 5 символов то слева добавиться пробелов до 5
print(d.ljust(5)) # Вывод 111__ , аналогично но пробелы с другой стороы
Также есть необязательный параметр, который указывает какой символ мы хотим вмемсто пробела
print(d.rjust(5,'s')) # Вывод ss111

Метод разделение строки по какому-то символу .split()
w = 'ivanov ivan ivanovich'
w.split() # ['ivanov', 'ivan' 'ivanovich'] # разделил строку по пробелу
w.split('n') # ['iva', 'ov iva', 'iva', 'ovich'] # разделили по 'n'

Метод соединения списка строк в одну строку .join()
s = ['ivanov', 'ivan', 'ivanovich']
print('!'.join(s)) # ivanov!ivan!ivanovich # не добавляет в конец

y = 'hello world'
print('!'.join(y)) # h!e!l!l!o! !w!o!r!l!d # просто вставит строку

Метод удаления пробелов .strip(), lstrip(), rstrip()
q = '   hello    '
print(q.strip()) # удаление пробелов слева и справа
print(q.lstrip()) # удаление пробела только слева
print(q.rstrip()) # удаляет пробелы справа
