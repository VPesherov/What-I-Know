Синтаксис срезы

stroka[start:stop:step]

start - индекс с которого начинается срез

stop - индекс на котором заканчивается срез, но не включая сам индекс stop

step - шаг с которым берутся буквы в срезе

Примеры:
word = "hello"

word[0:4]
# Вывод hell

Срез всей строки начиная с какого-то индекса
s[4:]
# начиная с 4го символа и до конца

s[:4]
# начиная от 4го символа и к началу

s[::2]
#взять все элементы через шаг 1

s[::-1]
