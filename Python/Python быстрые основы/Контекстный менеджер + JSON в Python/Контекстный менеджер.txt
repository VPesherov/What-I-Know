Контекстный менеджер - это специальная конструкция управления к ресурсу.

Менеджер контекста контролирует такие типы ресурса к которым нужно подключаться, а затем отключаться
и конкретно он контролирует что нужно делать когда мы получаем доступ к ресурсу и что нужно делать
когда этот доступ уже не нужен.

Пример:

Во время работы с файлом произошла ошибка и файл по итогу не закрылся, контекстный же менеджер
поможет в любом случае окончить работу с файлом даже при возникновение ошибок
Опишем контекстный менеджер

with open('file.txt', 'w') as file:
	file.write('123')
	file.write('hello')
	# как только отработает этот блок файл автоматически закроется

print('end')

Как доказательство что файл закрылся мы не сможем вне блока with добавить ещё что-то

with open('file.txt', 'w') as file:
	file.write('123')
	file.write('hello')
	# как только отработает этот блок файл автоматически закроется

file.write('text') # будет ошибка так как файл закрылся
print('end')
