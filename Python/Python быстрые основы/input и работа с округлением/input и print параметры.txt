чтоб ввести несколько чисел в одну строку
n1, n2, n3 = map(int, input().split()) #можно будет ввести три числа через пробел

Параметр sep - необязательный параметр который добавляет разделитель

print(1, 2, 3, 4, sep = '*')

Вывод
# 1*2*3*4

end - необязательный параметр которы добавляет в конец строки нужный символ и
не даёт перенестись printу

print(1,2,3, end='!')
print('Hello')

Вывод
# 1,2,3!Hello
