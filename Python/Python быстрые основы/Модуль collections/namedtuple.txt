Именованные кортежи - дают возможно дать именнование элементам которые они содержут
что бы проще было их искать и обращаться

Допустим есть список кортежей
players = [('Carlson', 1990, 2842), ('Caruana', 1992, 2822), ('Mamedyarov', 1985, 2801)]

И допустим мы бы хотели из кортежа получить имя игрока допустим
players[0].name # такое не сработает

Для этого пи приходит на помощь namedtuple

from collections import namedtuple

Player = namedtuple('Player', 'name age rating') # Нужно создать что-то типо класса где перечисляем
						# его название и список параметров
players = [Player('Carlsen', 1990, 2842), Player('Caruana', 1992, 2801)]

И выведем теперь то что получилось

print(players[0])
print(players[0].name)

# Вывод
Player(name='Carlsen', age=1990, rating=2842)
Carlsen
#

Готово! Теперь можно обращаться к нужному полю
