Counter(итерабельная_последовательность) - считает количество уникальных 
элементов в итерабельной последовательности

from collections import Counter

s = 'abracadabra'
print(Counter(s)) # Counter({'a': 5, 'b': 2, 'r': 2, 'c': 1, 'd': 1})

мы можем обойти все уникальный элементы

for i in letters():
	print(i)

# Вывод
a
b
r
c
d
#

но мы можем обойти их включая количество данных элементов

for i in letters.elements():
	print(i)

# Вывод
a
a
a
a
a
b
b
r
r
c
d
# вывелось по количеству

также можно отсортировать их по количеству, если в скобках указать число то он выведет 
заданное количество

letters.most_common() # [('a', 5), ('b', 2), ('r', 2), ('c', 1), ('d', 1)]
letters.most_common(3) # [('a', 5), ('b', 2), ('r', 2)]
