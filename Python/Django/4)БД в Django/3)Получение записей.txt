Команда
Movie.objects.all() # вернёт все записи в QuerySet 

Для понятности вывода, в модели можно использовать метод def __str__(self)

def __str__(self):
	return f'{self.name} - {self.rating} %'

Тогда эта команда вернёт записи вида указанного в фуннкции __str__

Также к Movie.objects.all()
можно применять срезы

например
Movie.objects.all()[:2] # выведет первые 2 записи
Также это работает как обращение по индексу
Movie.objects.all()[3] - к четвёртому элементу(нумеруется с нуля)

то есть мы можем сделать следующее

a = Movie.objects.all()[2]
и теперь можно обратиться к элементам 
a.name
a.rating
