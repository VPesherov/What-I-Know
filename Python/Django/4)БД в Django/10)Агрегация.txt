Агрегированние - объединение несколько элементов в единое целое
(то есть это функции из запроса например суммы, максимум, минимум, среднее и тд)

Например что-би найти сумму столбца

Movie.objects.all().aggregate(Sum('budget')) # результат вернётся в качестве словаря
# { 'budget__sum' : 7890352 }

Можно сразу агрегировать по нескольким полям

Movie.objects.all().aggregate(Sum('budget'), Avg('rating')) # вернётся аналогичный словарь
# {'budget__sum' : 832525,
  'rating__avg' : 67.356 }
