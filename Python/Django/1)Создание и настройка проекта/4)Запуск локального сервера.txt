1)У нас есть команда django-admin, по документации джанги
при создании проекта и работы с ним django-admin = manage.py

2)То есть можно прописать вместе django-admin - manage.py
Если вызывается ошибка, то либо путь до проекта указан неверно
Либо нужно запуск через питон
python3 manage.py
или
python manage.py

3)Затем пишем

python manage.py runserver - и запускается локальный веб севрер

4)Порт 8000 - это порт локальной машины по умолчанию

5)В консоли видны запросы пользователей

6)Ctrl + C - остановка сервера

7)Также при первом запуске создаётся БД (db.sqlite3)
