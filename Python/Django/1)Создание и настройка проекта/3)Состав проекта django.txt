В каждом Django проекте при его создании создаются определённые файлы:

1)manage.py -  с помощью строки execute_from_command_line, manage.py обрабатывает команды поступившие к нему из консоли

2)__init__ - пустой файл обозночающий что папка является пакетным модулем(озночает что содержимое этой папки
мы можем импортировать внутри нашего проекта) 

3)asgi и wsgi - они нужни для развёртки нашего готового проекта на каком-либо хосте

4)settings - настройка проекта(констант и тд)

5)url - настройка роутов, отвечает за перемещением по страницам сайта
