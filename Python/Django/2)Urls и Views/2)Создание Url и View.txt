Каждому Url соответсует какое-то view

Для обратки Url

1)Заходим в папку с названием проекта
2)Прописываем 
path('какой-то url', какое-то view)

3)Во view создаём функцию с названием нашего view
4)ПРАВИЛО
функция обязательно должна принимать в себя параметр request
Пример:

def leo(request):
	#код

5)И далее мы должны этот файл импортировать в urls
from horoscope.views import leo

или from horoscope import views

6)И во path('адррес', 'имя функции')
#имя функции без скобочек
path('horoscope/leo', leo)
