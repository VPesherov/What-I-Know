Используем когда не знаем сколько роутов будем использовать

Для этого роут надо сохранить в параметр 

Джанго понимает что это параметр если он обёрнут в <параметр>

Пример
path('<sign_zodiac>', views.get_info)

и тогда в функции

def get_info(request, sign_zodiac):
	#код
	return HttpResponse(f"Это знак {sign_zodiac}")
