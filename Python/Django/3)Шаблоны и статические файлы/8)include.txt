Include нужен для повторяющийся части типа верхнего меню или бокового меню, чтоб можно было
один раз его создать и просто подключать где надо

Например у нас есть html код состовляющий верхнее меню:

<nav>
    <a href="/horosope">Главная</a> |
    <a href="https://www.google.ru/">Гугл</a>
</nav>
 
и этот код должен быть у нас почти на всех страницах, и чтоб не пользоваться копированием
поместим этот код в отдельный html файл и подключим его к нужным страницам

Для этого в папке приложения в папке templates создадим папку includes и в ней создадим
нужный html файл например navbar.html и поместим туда код выше

Затем укажен где надо тег include

{% include 'includes/navbar.html' %}

Готово, также этот шаблон navbar имеет доступ ко всем переменным шаблона {{ переменная }}
НО если добавить в тег include слово only то переменные не будут доступны

{% include 'includes/navbar.html' only %}

или можно передать свои собственные переменные например

{% include 'includes/navbar.html' only with a=100 b=200 %}

и обратиться к ним {{ a }} и {{ b }}
