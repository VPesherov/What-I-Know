Когда у нас есть два очень похожих html файла у которых условно разница в нескольки строках
например в тайтле и немного в <body>, то можно создать базовый шаблон куда уже потом всё это передавать

Для этого
1)Создаём папку templates, создаём её на уровне проекта а не приложения
2)Создадим там html файл
3)Подключим эту папку с шаблонами к проекту
для этого в settings.py нашего проекта, в переменную TEMPLATES
в DIRS прописываем путь
Получится что-то типо этого

'DIRS': [
	f'{BASE_DIR}/templates'
],

4)Допустим у нас на двух страницах которые хотим изменить меняется только <title>
тогда прописываем в нашем шаблоне

{% block имя_блока %} {% endblock %}

5)Как прописали нужные блоки в которых будет меняться что-то
затем возвращаемся на наши похожие страницы и прописываем в самом вверху

{% extends 'имя_шаблона' %}

6)Прописываем значения которые хотим оставить на странице и оборачиваем их в аналогичные теги

{% block имя блока %}
Меню
{% endblock %}

пример в проекте my_django_project/my_page/
