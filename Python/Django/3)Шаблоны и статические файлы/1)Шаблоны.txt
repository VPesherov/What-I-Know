Шаблон - один из способов динамическрй генерации HTML.
Он состоит из обычного HTML файлика в котором содержится статическая часть HTML
и также дополнительных конструкций в которых описывается как будем гененрироваться
динамический контент вашего сайта

Создание шаблона
1)Переходим в папку приложения
2)Внутри него создаём папку templates
3)Внутри этой папке создаётся папка с названием приложения
4)И внутри уже этой папке создаём html файл

Затем нужно как-то найти наш файл из views и прочитать его
импортируем нужную функцию

from django.template.loader import render_to_string

и в нужном view прописываем
response = render_to_string('horoscope/info_zodiac.html') # сюда считается весь наш файл как строка
return HttpResponse(response)

и нужно подключить наше приложение, это можно сделать двумя способами
Более сложный способ
1)В settings.py находим переменную TEMPLATES
2)В 'DIRS' прописываем путь до нашей директории
3)'DIRS': [
            f'{BASE_DIR}/horoscope/templates'
        ],

Второй способ
1)Ищем переменную INSTALLED_APPS
2)И прописываем наше приложение дополнительно
3)'имя приложения' #его можно найти в переменной name в файле apps.py
